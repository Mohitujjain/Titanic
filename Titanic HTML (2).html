Data Description¶

1.Survived : People can survive or not. 2.Pclass : This is types of Ticket class ( 1 = 1st, 2 = 2nd, 3 = 3rd). 3.Sex : People are male or women. 4.Age : Age of People. 5.Sibsp : How many siblings and spouses aboard in the Titanic. 6.Parch : How many children and parents in the Titanic. 7.Ticket : Numbers in a Ticket. 8.Fare : Passengers fare. 9.Cabin : Cabin number of passengers. 10.Embarked : Port of Embarkation C = Cherbourg, Q = Queenstown, S =Southampton.
